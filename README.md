# careyshop-admin

> this is a CareyShop Admin for Vue.js project

```
# 克隆项目
git clone https://github.com/dnyz520/careyshop-admin.git

# 安装依赖
npm install
或
yarn

# 建议不要用cnpm安装 会有各种诡异的bug 可以通过如下操作解决 npm 下载速度慢的问题
npm install --registry=https://registry.npm.taobao.org

# 启动服务
npm run serve

# 正式发布
npm run build

# 检测修复
npm run lint

# 单元测试
npm run test:unit
```
