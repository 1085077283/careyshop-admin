# careyshop-admin

> this is a CareyShop Admin for Vue.js project

```

npm install
# or
yarn

# Compiles and hot-reloads for development
npm run serve

# Compiles and minifies for production
npm run build

# Lints and fixes files
npm run lint

# Run your unit tests
npm run test:unit
```
