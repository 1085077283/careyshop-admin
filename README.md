# careyshop-admin

> this is a careyshop admin for vue.js project

```
# 克隆项目
git clone https://github.com/dnyz520/careyshop-admin.git

# 安装依赖
npm install
或
yarn

# 启动服务
npm run serve

# 正式发布
npm run build

# 检测修复
npm run lint

# 单元测试
npm run test:unit
```

### 演示地址
demo地址：[https://demo.careyshop.cn/admin/](https://demo.careyshop.cn/admin/ "demo地址")
